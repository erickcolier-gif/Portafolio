## Descripcion

Portafolio profesional desarrollado desde cero con **HTML5, CSS3 y JavaScript vanilla**, sin frameworks ni dependencias externas. Disenado con un enfoque oscuro moderno (dark mode), animaciones fluidas y diseno completamente responsive para todos los dispositivos.

Refleja mi perfil como ingeniero especializado en la **digitalizacion y automatizacion de procesos**, con experiencia en el sector solar fotovoltaico y desarrollo de soluciones tecnicas con Python.

## Tecnologias

- **HTML5** — estructura semantica y accesible
- **CSS3** — variables CSS, Grid, Flexbox, animaciones y transiciones nativas
- **JavaScript (ES6+)** — Intersection Observer, animaciones de scroll, typing effect, particulas dinamicas
- **Google Fonts** — tipografia Inter

Sin librerias externas. Sin frameworks. Todo construido a mano.

## Como verlo localmente

```bash
# 1. Clonar el repositorio
git clone https://github.com/erickcolier-gif/Portafolio.git

# 2. Entrar a la carpeta
cd Portafolio

# 3. Abrir en el navegador
# Simplemente abre el archivo index.html en tu navegador
# o usa la extension Live Server en VS Code
```

No requiere instalacion de dependencias ni servidor backend.

## Funcionalidades destacadas

- **Navbar fija** con efecto blur al hacer scroll y menu hamburguesa en movil
- **Typing effect** con multiples titulos que rotan automaticamente
- **Particulas animadas** generadas dinamicamente en el hero
- **Animaciones de entrada** con Intersection Observer al hacer scroll
- **Barras de idiomas** animadas al entrar en vista
- **Formulario de contacto** con validacion y mensaje de confirmacion
- **100% responsive** desde 320px hasta pantallas 4K
